##### 全局

复制/粘贴：command + c/v

关闭当前页面：command + w

切换程序：command + tab

搜索：command + space（搜索term

home/end：ctrl + A/E  (fn + ←→)

切换输入法：ctrl + space

全屏：ctrl + command + f

最小化：command + m



##### chrome快捷键

前进后退：command + ←→

刷新：command + r





##### goland快捷键

设置：command + ,

前进后退：command + option + ←→

进入函数：command + b

Quick documentation：f1 / ctrl + j

Debug / run：ctrl + d / r
